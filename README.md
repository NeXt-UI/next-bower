# NeXt-bower
NeXt UI Toolkit `bower` and `npm` distribution. You can install this package either with `npm` or with `bower`.

# Install
### bower

```shell
bower install NeXt
```

Then add a `<script>` to your `index.html`:

```html
<script src="/bower_components/next-bower/js/next.js"></script>
```
